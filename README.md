# 1 — 프로젝트 소개

## 프로젝트명

여행 일정 공유 웹 서비스 (Trip Planner Web)

## 프로젝트 목적

- 여러 사용자가 함께 여행 일정을 계획하고 공유하는 과정에서 발생하는 일정 관리의 비효율 해결
- 대학생 및 소규모 그룹 사용자 대상 일정 관리 서비스 제공
- 메신저 및 문서 기반 일정 공유 방식의 한계 개선
- 날짜별 일정 정리 및 공동 편집 기능 제공

## 기대 효과

- 날짜별 여행 일정 체계적 관리 가능
- 참여자 간 일정 공유 및 협업 가능
- 지도 기반 위치 확인으로 이동 동선 파악 가능
- 일정 공유 링크 기능 제공
- MVC(Model2) 구조 및 DB 연동 이해

---

# 2 — 프로젝트 배경 및 범위

## 배경

- 메신저 및 문서 기반 일정 공유 시 정보 분산 문제 발생
- 일정 변경 시 실시간 반영 어려움 존재
- 여러 사용자가 하나의 일정에 동시에 참여 가능한 시스템 필요
- JSP / Servlet / JDBC 기반 MVC(Model2) 구조 적용

## 프로젝트 범위 (Scope)

### 포함 기능

- 사용자 회원가입 및 로그인 기능
- 여행 일정 생성 및 수정 기능
- 날짜별 여행 일정 등록 기능
- 일정 공유 링크 생성 기능
- 여행 참여자 초대 및 공동 편집 기능

### 제외 기능

- 고급 보안 인증 (OAuth 등)
- 다국어 지원 기능

## 사용 기술 스택

Java / JavaScript / HTML / CSS
JSP / Servlet / JDBC
MySQL / Tomcat
Bootstrap / Kakao Map API
Git & GitHub
