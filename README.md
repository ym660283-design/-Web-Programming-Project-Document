# 1 — 프로젝트 소개

## 프로젝트명

여행 일정 공유 웹 서비스 (Trip Planner Web)

## 프로젝트 목적

- 본 프로젝트는 여러 사용자가 함께 여행 일정을 계획하고 공유하는 과정에서 발생하는 일정 관리의 비효율성을 해결하기 위해 개발한다.
- 여행을 준비하는 대학생 및 소규모 그룹 사용자를 주요 대상 사용자로 설정한다.
- 기존에는 메신저나 문서로 여행 일정을 공유하는 경우가 많아 관리가 어렵다는 문제를 해결하고, 날짜별 일정 정리, 위치 확인, 공동 편집 기능을 통해 효율적인 일정 관리가 가능하도록 개선한다.

## 기대 효과

- 사용자들이 여행 일정을 날짜별로 체계적으로 관리하고 참여자 간 실시간으로 공유할 수 있다.
- 지도 기반 위치 표시 기능을 통해 여행 동선을 직관적으로 확인할 수 있다.
- 일정 공유 링크 기능을 통해 외부 사용자와도 쉽게 여행 계획을 공유할 수 있다.
- JSP/Servlet 기반 MVC(Model2) 구조 이해 및 데이터베이스 연동 능력을 향상시킬 수 있다.

# 2 — 프로젝트 배경 및 범위

## 배경

- 여행을 계획할 때 여러 사람이 메신저나 문서 파일을 통해 일정을 공유하는 경우 일정 관리가 어렵고 정보가 분산되는 문제가 발생한다.
- 본 프로젝트는 여러 사용자가 하나의 여행 일정에 함께 참여하고 날짜별 계획을 체계적으로 관리할 수 있도록 지원하는 웹 서비스를 구현하는 것을 목표로 한다.
- 본 시스템은 JSP / Servlet / JDBC 기반 MVC(Model2) 구조를 적용하여 사용자 관리, 일정 관리, 데이터베이스 연동 기능을 통합적으로 구현한다.

## 프로젝트 범위 (Scope)


| 포함 (In-Scope)                    | 제외 (Out-of-Scope)      |
| ------------------------------------ | -------------------------- |
| 사용자 회원가입 및 로그인 기능     | 고급 보안 인증(OAuth 등) |
| 여행 일정 생성 및 수정 기능        | 다국어 지원 기능         |
| 날짜별 여행 일정 등록 기능         |                          |
| 일정 공유 링크 생성 기능           |                          |
| 여행 참여자 초대 및 공동 편집 기능 |                          |

## 사용 기술 스택

- 언어: Java / JavaScript / HTML / CSS
- 서버: Apache Tomcat
- 백엔드: JSP / Servlet
- 데이터베이스: MySQL (JDBC)
- 프론트엔드: Bootstrap
- 지도 API: Kakao Map API
- 버전 관리: Git & GitHub
